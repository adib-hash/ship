What you'll need

A few things to have ready before Day 1

The program is designed to be genuinely practical, which means we work with real tools from the first session. To keep up and get the most out of each week, every student will need three things in place before the cohort starts.

A capable laptop

Any modern computer from 2020 or newer with at least 8GB of RAM — Mac or Windows. As a reference point, an M1 MacBook Pro with 8GB RAM runs everything in the curriculum comfortably.

Claude Pro ($20/mo)

A Claude Pro subscription gives you meaningfully higher usage limits than the free plan, and we'll push those limits regularly. You can subscribe for the two months of the program and cancel afterward — and learning how token consumption and model usage works is itself part of the curriculum.

$5 in Claude API credits

In Week 5, we build AI-powered features using the Claude API directly. You'll add $5 in prepaid API credits to an Anthropic account — plenty for everything we do in class, with room to experiment on your own afterward.

Total out-of-pocket: $50 (program) + ~$40 (two months of Claude Pro) + $5 (API credits) = roughly $95 for the full experience. We'll confirm you have all three in place before finalizing your spot.

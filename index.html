The curriculum

Six sessions. One real deliverable each week.

Each session pairs one "think & plan" skill with one "build & ship" deliverable — so you leave every session with something real, not just notes.

Session 1

AI Fluency + Your First Website

You ship: A live professional website at your own domain

The AI landscape, the analyst mental model for prompting, iterating to the truth, buying a domain, push-to-deploy with GitHub + Vercel. Ship something to the internet on day one.

Session 2

Planning Like a Builder + Interactive Apps

You ship: A work tool saved to your phone home screen

Writing a PRD before you build (same discipline as a precise brief), the CLAUDE.md file, having AI interview you before writing a line of code, mobile-first UI, dark mode — saved as a PWA to your home screen.

Session 3

Working With APIs

You ship: A data-driven app connected to live information

What APIs are (requests, responses, keys — in plain English), securely storing secrets, displaying live data, and a debugging playbook: reading error messages, using dev tools, diagnosing CORS errors and rate limits methodically.

Session 4

Full-Stack Apps With Supabase

You ship: An app with users, accounts, and saved data

Give your apps memory. Database tables and relationships without SQL expertise, user authentication, reading/writing data from the front end. The difference between a toy and a real product.

Session 5

Claude Code, the Claude API + Claude Skills

You ship: An AI-powered feature + a reusable Claude Skill

A professional developer workflow with Claude Code CLI, making targeted edits in a real codebase, the Claude API (summarization, Q&A over custom content, structured extraction), and packaging repeatable workflows into Claude Skills you trigger with one command.

Session 6

Shipping a Native iOS App + Demo Day

You ship: A native iOS app running on your phone

Capacitor wraps your web app for native iOS — no App Store gatekeeper required. App icons, splash screens, TestFlight for sharing with colleagues. Plus a two-minute demo to the cohort — because shipping without showing it to someone doesn't count.

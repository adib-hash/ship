# Ship — ship.ihsan.build

Landing page for **Ship**, a 6-week cohort program that teaches 7th–12th graders how to design, build, and ship real software with AI as a collaborator.

Static single-file site. No build step.

- `index.html` — the landing page (with For Students / For Parents toggle)
- `ship-ihsan-curriculum-v3.md` — full curriculum
- `ship-ihsan-onepager.md` — marketing one-pager

## Local preview

```
open index.html
```

## Deploy

Pushed to `main` → Vercel auto-deploys → served at `ship.ihsan.build`.
